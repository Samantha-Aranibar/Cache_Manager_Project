# Data Structures in C++

A semester-long C++ project implementing fundamental data structures from scratch. Each milestone introduces a new data structure and reinforces concepts such as dynamic memory management, object-oriented programming, algorithm design, and efficient data organization.

---

## Project Overview

This project was completed as part of a university Programming Methodology course. Rather than relying on the C++ Standard Template Library (STL), each data structure was implemented manually to better understand its internal behavior and performance.

The project progresses through four milestones:

- Doubly Linked Lists
- Hash Tables
- Cache Manager
- Binary Search Trees

Each milestone builds upon concepts learned in previous milestones.

# Milestones

## Milestone 1 – Doubly Linked List

Implemented a custom doubly linked list supporting insertion, deletion, traversal, and node manipulation.

Topics:

- Dynamic memory
- Pointers
- Linked Lists

---

## Milestone 2 – Hash Table

Implemented a hash table supporting insertion, deletion, lookup, and collision management.

Topics:

- Hashing
- Fast searching
- Constant-time lookup

---

## Milestone 3 – Cache Manager

Built a FIFO Cache Manager using the custom linked list and hash table implementations.

Additional work included:

- UML Class Diagram
- UML Sequence Diagram

Topics:

- Software Design
- Data Structure Integration
- Cache Management

---

## Milestone 4 – Binary Search Tree

Implemented a Binary Search Tree supporting insertion, deletion, searching, traversal, and recursive algorithms.

Topics:

- Trees
- Recursion
- Tree Traversals
