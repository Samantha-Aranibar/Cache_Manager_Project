# Milestone 1 – Doubly Linked List

## Overview

This milestone implements a doubly linked list from scratch in C++. The implementation supports insertion, deletion, node movement, traversal, and list management without using STL containers.

## Features

- Insert at head
- Insert at tail
- Remove node
- Remove head
- Remove tail
- Move node to head
- Move node to tail
- Clear list
- Print list
- Reverse print list

## Concepts

- Dynamic memory allocation
- Pointers
- Doubly linked lists
- Object-Oriented Programming
