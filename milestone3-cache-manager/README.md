# Milestone 3 – Cache Manager

## Overview

This milestone combines the linked list and hash table implementations into a FIFO Cache Manager capable of storing and efficiently retrieving cached entries.

## Features

- FIFO Cache
- Automatic Cache Eviction
- Cache Lookup
- Insert
- Remove
- Clear Cache
- Print Cache
